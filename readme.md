# nodecg-sample-vue-parcel

Sample [NodeCG](https://nodecg.com/) bundle using [Vue](https://vuejs.org/) and [Parcel](https://parceljs.org/).

## Usage

1. Install this bundle to NodeCG context

2. Install dependencies
```
$ yarn install
```

3. Bundle source code
```
$ yarn build
```

4. Start NodeCG
```
$ yarn start
```
